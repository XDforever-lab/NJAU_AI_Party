# NJAU_AI_Party - 南农大先生跨时空对话

## 项目介绍
这是一个基于Streamlit + Kimi大模型的互动应用，支持与南京农业大学三位著名学者（金善宝、樊庆笙、万国鼎）进行“跨时空对话”，同时展示先生们的红色档案与贡献。

## 环境要求
- Python 3.8 及以上版本

## 安装与运行步骤
1. 克隆本仓库到本地：
    ```bash
    git clone https://github.com/3125189495-lab/NJAU_AI_Party.git
    ```
2. 进入项目目录：
    ```bash
    cd NJAU_AI_Party
    ```
3. 安装依赖库：
    ```bash
    pip install -r requirements.txt
    ```
4. 启动应用：
    ```bash
    streamlit run app.py
    ```

## 使用说明
- 启动后，在浏览器打开终端显示的 URL（如 http://localhost:8501）；
- 左侧侧边栏选择要对话的 “南农大先生”；
- 输入 Moonshot（Kimi）的 API Key（需从月之暗面官网注册获取）；
- 可通过 “查看先生的红色档案” 模块，了解先生们的贡献。

## 注意事项
- API Key 需自行注册获取，且确保网络可访问 Moonshot API 服务；

- 项目中的图片素材为南农大相关资料，仅供学习展示使用。
